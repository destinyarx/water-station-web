# AI Handoff Documentation — AquaFlow (Water Station Web)

This is the entry point for future AI coding agents working on this repository.

## Project summary

AquaFlow is a multi-tenant Next.js/Supabase/Clerk SaaS dashboard for small
**water refilling station** businesses (Philippines context). It manages
customers, product/refill catalogs, deliveries, expenses, and equipment
maintenance for two roles — Owner and Staff — each scoped to one
organization via `org_id`.

## Documentation purpose

These 16 files let a future AI agent understand, extend, or port this
codebase without repeatedly re-scanning the whole repository. They were
generated by inspecting specs (`docs/specs/`), ADRs (`docs/adr/`), existing
project docs, and source code — not by guessing. Every non-trivial claim is
labeled **Confirmed**, **Inferred**, **Unknown**, **Potentially outdated**,
or **Requires validation**.

## Important limitations of this analysis

- This is a **static documentation pass**, not a live-environment audit. Several claims are marked **Requires validation** because they depend on the live Supabase project's actual schema/RLS state, which cannot be inspected from the repository alone (no `supabase/migrations` folder exists).
- `docs/12-testing-and-operations.md` records real command output (`npm run test`, `npm run lint`, `npx tsc --noEmit` — all passed) but `npm run build`/`npm run dev` were not executed (side-effect risk, out of scope for a documentation pass).
- Supabase Edge Function source code (org creation, add-staff, an internal email tool) lives outside this repository and could not be inspected — only their client-side contract is documented.
- No security penetration test was performed; [`10-security-and-risks.md`](10-security-and-risks.md) is explicit that absence of a finding is not proof of safety.
- This package was produced by six parallel research passes plus a synthesis/consistency pass; cross-references between files were checked, but a residual inconsistency is possible — treat repository code as the final source of truth when in doubt, per the instructions in `14-ai-agent-context.md`.

## Areas requiring owner confirmation
